# 酒店管理系统 java1156

## 项目概述

酒店管理系统是一款基于Java和Vue的前后端分离的B/S架构系统。该系统采用MVC模式设计，结合SpringBoot和Mybatis等现代技术栈，实现了用户与管理员之间的功能分离与高效互动。

## 技术栈

- **架构**: B/S、MVC
- **系统环境**: Windows/Mac
- **开发环境**:
  - IDEA
  - JDK 1.8
  - Maven
  - Mysql
- **技术栈**:
  - Java
  - Mysql
  - SpringBoot
  - Mybatis
  - Vue

## 功能模块

### 用户模块

- 用户登录
- 用户注册
- 预约住房
- 留言
- 查看公告
- 查看预约
- 入住查询

### 管理员模块

- 管理员管理
- 用户管理
- 房型管理
- 房间管理
- 预约管理
- 入住管理
- 留言管理
- 公告管理

## 系统角色

- 用户：可登录注册，进行住房预约和留言等操作。
- 管理员：负责系统管理，包括用户、房型、房间、预约、入住和公告等方面的管理。

## 运行环境

- 服务器：支持Windows和Mac操作系统
- 数据库：Mysql
